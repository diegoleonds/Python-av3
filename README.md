## Sobre o projeto

Sistema Python para listar registros em uma página web.

## Instruções

Digite os seguintes comandos no terminal:

### Para rodar

```
python -m flask run
```

### Para adicionar um registro
```sql
sqlite3 database.db
INSERT INTO contato VALUES(1, "Nome", "Sobrenome", "email@email.com");
```
